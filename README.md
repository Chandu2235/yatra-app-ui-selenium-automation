# YatraApplicationAutomationPractice

Learning repository for practicing Page Object Model (POM) structure and automation framework design along with CI/CD intergations pipelines. 

This project demonstrates an end-to-end (e2e) automation framework for the Yatra web application, built using a combination of Page Object Model (POM) approaches. The framework is designed to enhance test maintainability, reusability, and reporting capabilities.

Technologies Used:
Java | Selenium WebDriver | TestNG | Maven | GitHUB | Jenkins | HTML/JS/CSS(DOM Structure Understanding) | Debugging
